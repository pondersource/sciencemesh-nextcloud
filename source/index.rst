Technical / Developer Documentation
===================================
The 'sciencemesh' app for Nextcloud adds API endpoints so that Nextcloud
can serve as a user backend, a shares backend, and a storage backend for revad.

Installation Guide
==================
Install revad next to your Nextcloud instance,
install the ScienceMesh app on your Nextcloud instance,

User Documentation
==================
The ScienceMesh app gives a GUI where the user can generate and paste OCM invitations.
When searching for a collaborator to share with, remote ScienceMesh users who
accepted your invitation will show up.

Admin Documentation
===================
Configure "iopUrl" to point to your revad instance.
